🔌 API Docs

AppTrust REST API

Integrate AppTrust's powerful scam detection directly into your app or platform. Simple REST API, JSON responses.

POST /analyze — Analyze an App

POST /analyze Content-Type: application/json { "package_id": "com.whatsapp" }

POST /search — Search Apps

POST /search Content-Type: application/json { "query": "WhatsApp" }

Response Format

{ "success": true, "title": "WhatsApp", "trust_score": 9, "label": "Safe", "fake_pct": 4, "avg_polarity": "0.81", "permissions": [...], "reviews": [...] }

About AppTrust

AppTrust was built to protect everyday users from malicious apps that steal data, extort money, or compromise privacy. We believe everyone deserves to know if an app is safe before installing it.

🎯

Our Mission

Make app safety accessible to everyone. No technical knowledge required — just search and get answers instantly.

🤖

Our Technology

Advanced NLP and machine learning models trained on millions of apps, reviews, and threat intelligence data.

🔒

Privacy First

We never store personal data. All scans are anonymous. Your searches stay private — always.

Built in Maharashtra, India 🇮🇳

AppTrust was developed with a focus on protecting users in India, where loan app scams and fraudulent applications have caused significant harm to thousands of families. Our detection models are specifically tuned for apps targeting Indian users.
